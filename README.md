# WebWaka Core Dashboard Control

**Status:** Core Module (Substrate)  
**Class:** `core`  
**Version:** `0.1.0`

## Installation

```bash
npm install @webwaka/core-dashboard-control
```

**Note**: This package is published to npm registry. GitHub-based installs are no longer supported.


## Purpose

Dashboard lifecycle and privilege-driven UI control for the WebWaka Modular Ecosystem.

## Responsibilities

This module provides the foundational infrastructure for:

- Dashboard registration and lifecycle management
- Privilege-driven UI composition and rendering
- Dashboard access control and entitlement verification
- Dashboard state management and persistence
